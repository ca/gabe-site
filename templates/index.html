Projects

X-Spark



X-Spark is a 4 player cooperative shooter, where players fend off an alien invasion. The game features a mechanic in which enemies are only vulnerable to a specific player at once. Players must coordinate their attacks in order to defeat alien mother ship.

View

Colors



Colors is a casual puzzle game for mobile. Players must solve different puzzles by dragging and combining colored tiles to gain points.

My role in this project was lead designer / programmer. In the Unity project, I built the color mixing mechanic. I also rebuilt the application in Swift for iOS devices.

View

Smashcii



Smashcii is a fast paced rogue-like shooter. In each level, the player must survive waves of enemies, to improve his or her score. Each level provides different pathways to avoid and outwit pursuing enemies.

View

Collision Detection



This is a project I built to simulate physical collisions in 3D between different types of geometric primitives. The simulation incorporates linear and angular momentum in order to calculate collision response between spheres and cubes.

View

Platformer



This project was built as a proof of concept for a platformer where you control the platforms instead of the character. It utilizes integration techniques to simulate velocity and acceleration due to gravity, as well as collision detection and response for a sphere and line segment.
