Windows Game

Platformer

This project was built as a proof of concept for a platformer where you control the platforms instead of the character. It utilizes integration techniques to simulate velocity and acceleration due to gravity, as well as collision detection and response for a sphere and line segment.

This was built in OpenGL and C++ using Visual Studio.
