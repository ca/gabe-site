Multi Platform Game

X-Spark

X-Spark is a 4 player cooperative shooter, where players fend off an alien invasion. The game features a mechanic in which enemies are only vulnerable to a specific player at once. Players must coordinate their attacks in order to defeat alien mother ship.

X-Spark was built using Unreal Engine.

As designer / programmer on this project, I implemented color based vulnerability of enemies and obstacles. I also built blueprints responsible for character and barrier spawning, input handling, projectile firing and response.
